# Sistema de Gerenciamento de Ninjas

Aplicação Java (console) para gerenciamento de uma lista dinâmica de ninjas,
utilizando Collections e algoritmos de ordenação e busca.

## Objetivo
Praticar LinkedList, POO e algoritmos eficientes de busca e ordenação.

## Tecnologias
- Java SE
- LinkedList (Collections Framework)
- Comparator
- Busca binária

## Funcionalidades
- Cadastro e remoção de ninjas
- Listagem e acesso por posição
- Ordenação por nome, idade ou vila
- Busca eficiente por nome

## Estrutura
Main.java  
Sistema.java  
Ninja.java  

## Execução
Execute a classe Main e utilize o menu no console.
